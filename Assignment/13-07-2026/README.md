📚 Topics Covered
🔹 1. SELECT
The SELECT statement is used to retrieve data from one or more columns in a table. It is the most fundamental and frequently used SQL command.

✅ Concepts Practiced
📄 Selecting all columns
🎯 Selecting specific columns
🔍 Retrieving required information
🔹 2. WHERE
The WHERE clause is used to filter records based on specified conditions.

✅ Concepts Practiced
🔎 Filtering rows
🔢 Numeric comparisons
📝 Text comparisons
⚡ Multiple conditions
🔹 3. GROUP BY
The GROUP BY clause groups rows that have the same values into summary rows.

✅ Concepts Practiced
➕ SUM()
📊 AVG()
🔢 COUNT()
⬆️ MAX()
⬇️ MIN()
🔹 4. HAVING
The HAVING clause filters grouped data after the GROUP BY operation.

✅ Concepts Practiced
🎯 Filtering aggregated results
📈 Using aggregate functions with conditions
🔹 5. TOP
The TOP clause limits the number of rows returned by a query.

💡 Note: TOP is supported in SQL Server. In MySQL and PostgreSQL, use LIMIT.

✅ Concepts Practiced
🥇 Top N records
💰 Highest salary
⭐ Highest experience
📑 Ordered results
🔹 6. DISTINCT
The DISTINCT keyword removes duplicate values and returns only unique records.

✅ Concepts Practiced
🏢 Unique departments
🌍 Unique cities
🧩 Unique combinations of columns
🔹 7. Comparison Operators
Comparison operators are used to compare values in SQL conditions.

✅ Operators Covered
=
>
<
>=
<=
<> (Not Equal)
📌 Used for comparing values and filtering records.

🔹 8. Logical Operators
Logical operators combine multiple conditions within a query.

✅ Operators Covered
🤝 AND
🔀 OR
🚫 NOT
📌 Used to build complex filtering conditions.

🔹 9. IN & NOT IN
The IN operator checks whether a value exists within a list of values.

✅ Concepts Practiced
📋 Multiple value filtering
❌ Excluding multiple values
🔹 10. BETWEEN
The BETWEEN operator filters values within a specified range.

✅ Concepts Practiced
📏 Numeric ranges
✅ Inclusive range filtering
🚫 NOT BETWEEN
🔹 11. LIKE Operator
The LIKE operator performs pattern matching on text values using wildcards.

✅ Wildcards Covered
% ➜ Zero or more characters
_ ➜ Exactly one character
🔍 Concepts Practiced
Names starting with a character
Names ending with a character
Names containing a character
Pattern matching using wildcards
