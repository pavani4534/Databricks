📚 Concepts Covered 🔹 HAVING Clause Used with GROUP BY to filter grouped results Difference: WHERE → filters rows HAVING → filters groups

🔹 ORDER BY Sorting results: Ascending (ASC) Descending (DESC) Sorting using multiple columns

🔹 JOIN Operations Combining data from multiple tables Types of joins: INNER JOIN LEFT JOIN RIGHT JOIN Understanding relationships between tables

🔹 Subqueries (Nested Queries) Query inside another query Used for: Comparisons Filtering Finding MAX, AVG, etc.

🔹 Correlated Queries Subqueries that depend on the outer query Executed row-by-row

🔹 Combined Queries Mixing: JOIN GROUP BY HAVING Subqueries Used to solve real-world complex problems
