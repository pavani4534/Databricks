# MySQL Numeric Functions

This repository contains practice topics and examples for **Numeric Functions in MySQL** using an employee salary database.

These functions are commonly used in:

* Salary calculations
* Financial reports
* Mathematical operations
* Decimal handling
* Data formatting
* Business analytics

---

# Table Used

## employee_salary

The table contains:

* Employee ID
* Employee Name
* Base Salary
* Bonus
* Tax Percentage
* Experience Years

---

# Topics Covered

## 1. ABS() – Absolute Value

* Convert negative numbers into positive values

## 2. ROUND() – Round Numbers

* Round decimal values to specified digits

## 3. CEIL() / CEILING() – Round Up

* Round values upward to nearest integer

## 4. FLOOR() – Round Down

* Round values downward to nearest integer

## 5. TRUNCATE() – Remove Decimals

* Remove decimal values without rounding

## 6. MOD() – Remainder

* Find remainder after division

## 7. POWER() / POW() – Exponent

* Calculate powers of numbers

## 8. SQRT() – Square Root

* Find square root values

## 9. SIGN() – Sign Detection

* Identify positive, negative, or zero values

## 10. RAND() – Random Numbers

* Generate random decimal numbers

## 11. FORMAT() – Number Formatting

* Format numeric values with commas and decimal places

## 12. GREATEST() – Maximum Value

* Find the highest value among expressions

## 13. LEAST() – Minimum Value

* Find the smallest value among expressions
