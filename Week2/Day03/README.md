# MySQL String Functions

This repository contains examples and practice topics for commonly used **String Functions in MySQL**.

String functions are useful for:
- Text formatting
- Data cleaning
- Searching text
- Extracting characters
- Combining strings
- Handling NULL values
- Real-time text processing

# String Functions Covered

## 1. LENGTH() / CHAR_LENGTH()

Returns the number of characters in a string.

### Topics:
- Character counting
- String length calculation
- Multi-byte character handling

---

## 2. UPPER() / LOWER()

Converts text into uppercase or lowercase.

### Topics:
- Text formatting
- Standardizing data
- Case conversion

---

## 3. TRIM() / LTRIM() / RTRIM()

Removes unwanted spaces from strings.

### Topics:
- Leading spaces
- Trailing spaces
- Data cleaning

---

## 4. CONCAT()

Combines multiple strings into one.

### Topics:
- String joining
- Dynamic text generation

---

## 5. CONCAT_WS()

Combines strings using a separator.

### Topics:
- Custom separators
- Formatted output

---

## 6. SUBSTRING() / SUBSTR() / MID()

Extracts part of a string.

### Topics:
- Partial text extraction
- String slicing

## 7. LEFT() / RIGHT()

Extracts characters from the left or right side.

### Topics:
- Prefix extraction
- Suffix extraction

## 8. INSTR()

Returns the position of a substring.

### Topics:
- Searching text
- Finding special characters

## 9. LOCATE()

Finds the position of a substring with more flexibility.

### Topics:
- String searching
- Pattern location

## 10. REPLACE()

### Topics:
- Text replacement

### Topics:
- Reverse text operations
